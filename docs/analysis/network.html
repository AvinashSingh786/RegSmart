Network Analysis


Network analysis revolves around the network cards and connection that the machine has from wireless to wired. Information that was extracted from the registry are: network cards, intranet, wireless.

Wireless - in specific WiFi networks was looked into because it is the most common wireless network. Information extracted for this involves Name of the network, date of first connection, date of last connection, as well as the ID that Windows assigns.

For this the following registry keys were used to extract the information from SOFTWARE hive file: 

    Microsoft\\Windows NT\\CurrentVersion\\NetworkCards
    Microsoft\\Windows NT\\CurrentVersion\\NetworkList\\Nla\\Cache\\Intranet
    Microsoft\\Windows NT\\CurrentVersion\\NetworkList\\Nla\\Wireless
    Microsoft\\Windows NT\\CurrentVersion\\NetworkList\\Profiles
