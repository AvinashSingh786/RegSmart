Reporting


Reporting in RegSmart is simple, there are different options that can be used in reporting from choosing what types of analysis you wish as well as choosing the exclusion databases. Exclusion databases acts like a filter, that basically just omits non-important information for example common operating system services that will constantly be there. This is only for reporting that these databases are used, and its sole purpose is to reduce the length of the generated report.

Here is the list for the various options you have for generating a report:

  • System Analysis
  • OS Analysis
  • Application Analysis
  • Network Analysis
  • Device Analysis
  • ShimCache Analysis
