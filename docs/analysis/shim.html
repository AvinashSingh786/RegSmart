ShimCache Analysis


ShimCache analysis revolves around the cache information stored in the registry regarding Windows application compatibility etc. This was developed by https://github.com/mandiant/ShimCacheParser, and was incorporated into RegSmart as is with not modifications. RegSmart just calls this module in the background to generate the information provided by this module.

For this the following registry keys were used: 

    HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatibility\AppCompatCache
    HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache\AppCompatCache
