System Analysis


System analysis revolves around the system configurations and settings that have been set by the user for example environment variables, etc. Such information may be useful and show installed program traces that may have been uninstalled but still configured as a system path.

What this form analysis does it extract the computer name, operating system distribution as well as version, the last shutdown time as well as the process architecture and core count. Services are also extracted from the registry, but unfortunately no mapping of them due to time constraints, but maybe soon. It is simply a list of services stored in registry which shows a lot about the system configurations and setup.

The following keys were used for the extractions from the system and software hive:

    ControlSet001\\Control\\Session Manager\\Environment
    ControlSet001\\Control\\ComputerName\\ComputerName
    ControlSet001\\Control\\Windows
    ControlSet00\\Services

    Microsoft\\Windows NT\\CurrentVersion
