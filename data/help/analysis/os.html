OS Analysis


OS analysis revolves around the operating system installed on the machine as well as the users and operating specific identities. Information that was extracted from the registry are: product name, release id, current build, product id, path name, installed date, organization information, owner name.

User profiles - this is what will be used for the mapping and user profiling in future works. For now this just shows the username, SID, and the path to where the NTUSER.DAT file can be found for that user. This will help retriving the correct files and make it easier to organize and maintain user profiling.

User Accounts - based on the SAM entries for the user accounts created on the machine.

For this the following registry keys were used to extract the information from SOFTWARE and SAM hive files: 

    Microsoft\\Windows NT\\CurrentVersion
    Microsoft\\Windows NT\\CurrentVersion\\ProfileList
    Microsoft\\Windows NT\\CurrentVersion\\ProfileList\\{each SID} - to get the path for ntuser.dat for a user
    SAM\\Domains\\Account\\Users\\Names
